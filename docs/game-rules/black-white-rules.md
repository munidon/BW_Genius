# 더 지니어스 - 흑과 백 데스매치 규칙서

## 목적
이 문서는 1:1 게임 "흑과 백"의 프론트엔드/백엔드 구현 기준 규칙이다.

## 기본 구성
1. 각 플레이어는 1부터 9까지 숫자 타일 9장을 가진다.
2. 타일 색상은 다음과 같이 고정된다.
   - 흑색: 2, 4, 6, 8
   - 백색: 1, 3, 5, 7, 9

## 라운드 진행
1. 1라운드 선 플레이어는 코인 플립으로 선출된 플레이어가 선택한다.
2. 2라운드부터 선 플레이어는 직전 라운드 승자다.
3. 직전 라운드가 무승부이면, 직전 라운드의 선 플레이어가 다시 선 플레이어다.
4. 선 플레이어가 타일 1장을 뒷면으로 제출한다.
5. 후 플레이어가 타일 1장을 뒷면으로 제출한다.
6. 시스템(딜러)만 두 타일 숫자를 확인한다.
7. 승패는 숫자 비교로 정하고, 승자는 승점 1점을 얻는다.
8. 예외 규칙: 숫자 1은 숫자 9를 이긴다.
9. 라운드 결과 공개 시, 승/패/무만 공개된다. 양측이 낸 정확한 숫자는 상대에게 공개되지 않는다.

## 게임 종료
1. 9라운드 종료 후 더 높은 승점을 가진 플레이어가 승리한다.
2. 누군가 5승을 먼저 달성하면 즉시 게임이 종료된다.
3. 9라운드 종료 시 동점이면 무승부로 종료한다.

## 구현 시 주의사항
1. 각 플레이어는 본인이 낸 타일 히스토리는 볼 수 있어야 한다.
2. 상대 타일의 정확한 숫자 히스토리는 볼 수 없어야 한다.
3. 공개 가능한 정보는 라운드 결과, 점수, 선/후 플레이어 정보, 사용/미사용 타일 상태다.
4. 타일 선택은 이미 사용한 타일을 다시 제출할 수 없도록 강제한다.
5. 플레이 중 호스트 또는 게스트가 `5분` 동안 서버가 감지한 활동이 없으면 잠수로 판정한다.
6. 잠수 판정된 플레이어는 `기권 후 나가기`를 누른 것과 같은 처리로 즉시 패배한다.
7. 남아 있는 플레이어는 즉시 승리 처리되고 종료 화면으로 진입해야 한다.
8. 이때 남아 있는 플레이어에게는 실제 떠난 플레이어 닉네임을 넣은 `@@님이 떠나 게임을 종료합니다` 문구를 보여 준다.
9. 화면 재접속과 heartbeat는 잠수 타이머를 리셋하며, 단순 화면 조회는 리셋하지 않는다.
