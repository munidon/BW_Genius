# 십이장기 구현 기준 규칙서

## 목적
이 문서는 1:1 실시간 게임 "십이장기"의 프론트엔드/백엔드 구현 기준 규칙이다.

## 보드 구성
1. 게임 판은 총 12칸으로 구성된다.
2. 사용자 요청에는 `가로 4칸, 세로 3칸`과 `각 플레이어 앞쪽 3칸이 진영`이 함께 적혀 있어 해석 충돌이 있다.
3. 이 문서는 진영 3칸 조건에 맞춰 우선 `3열 x 4행` 보드를 기준안으로 사용한다.
4. 화면 기준으로 위쪽 1행은 상단 플레이어의 진영, 아래쪽 4행은 하단 플레이어의 진영이다.
5. 각 진영은 색을 조금 다르게 두어 구분한다.

## 초기 배치
1. 각 플레이어는 장, 상, 왕, 자를 1개씩 가진 상태로 시작한다.
2. 하단 플레이어 기준 초기 배치는 다음과 같다.
   - 4행 1열: 상
   - 4행 2열: 왕
   - 4행 3열: 장
   - 3행 2열: 자
3. 상단 플레이어는 하단 플레이어와 마주 보도록 대칭 배치한다.
   - 1행 1열: 장
   - 1행 2열: 왕
   - 1행 3열: 상
   - 2행 2열: 자
4. 말의 이동 방향은 각 말의 소유자 기준으로 고정되며, 게임 도중 회전하지 않는다.

## 말 이동 규칙
1. 모든 말은 한 번에 1칸만 이동한다.
2. 장은 앞, 뒤, 좌, 우로 이동할 수 있다.
3. 상은 대각선 4방향으로 이동할 수 있다.
4. 왕은 앞, 뒤, 좌, 우, 대각선까지 모든 방향으로 이동할 수 있다.
5. 자는 오직 자신의 전방으로만 이동할 수 있다.
6. 자가 상대 진영에 들어가면 즉시 후로 승격된다.
7. 후는 자신의 뒤 대각선 2방향을 제외한 6방향으로 이동할 수 있다.

## 잡기와 포로
1. 자신의 턴에는 말 1개를 이동하거나, 잡아둔 포로 1개를 내려놓을 수 있다.
2. 상대 말이 있는 칸으로 이동하면 해당 말을 잡는다.
3. 잡은 말은 포로가 되어 자신의 말로 전환된다.
4. 잡은 후를 다시 사용할 때는 자 상태로 뒤집어서 사용해야 한다.
5. 잡은 말을 사용할 때의 이동 방향 기준은 현재 소유자 기준으로 다시 적용된다.

## 포로 내려놓기 규칙
1. 포로를 내려놓는 행동도 턴 1회를 소모한다.
2. 이미 말이 놓여 있는 칸에는 포로를 내려놓을 수 없다.
3. 상대 진영에는 포로를 내려놓을 수 없다.
4. 잡은 말은 원하는 턴에 자유롭게 사용할 수 있으며, 사용하지 않고 보유해도 된다.

## 턴 진행
1. 선 플레이어부터 시작한다.
2. 매 턴 가능한 행동은 `말 이동` 또는 `포로 내려놓기` 중 하나다.
3. 사용자 요청에는 선공 결정 방식이 명시되어 있지 않으므로, 구현 계획에서는 게임 시작 시 랜덤 선공을 기준안으로 둔다.

## 승리 조건
1. 상대 왕을 잡으면 즉시 승리한다.
2. 자신의 왕이 상대 진영에 들어간 뒤, 상대 턴이 끝나고도 살아남아 자신의 다음 턴 시작 시점까지 버티면 승리한다.

## 구현 시 주의사항
1. 승격은 자가 이동을 마친 칸이 상대 진영일 때 즉시 반영한다.
2. 후를 잡으면 자로 되돌린 뒤 포로 목록에 넣는다.
3. 말 회전 기능은 제공하지 않는다.
4. 사용자 요청에 없는 추가 금수 규칙은 v1에 넣지 않는다.
5. 반복 수, 무승부, 시간 제한은 별도 정의가 없으므로 구현 계획에서 추가 정책 결정이 필요하다.
6. 플레이 중 호스트 또는 게스트가 `5분` 동안 서버가 감지한 활동이 없으면 잠수로 판정한다.
7. 잠수 판정된 플레이어는 `기권 후 나가기`를 누른 것과 같은 처리로 즉시 패배한다.
8. 남아 있는 플레이어는 즉시 승리 처리되고 종료 화면으로 진입해야 한다.
9. 이때 남아 있는 플레이어에게는 실제 떠난 플레이어 닉네임을 넣은 `@@님이 떠나 게임을 종료합니다` 문구를 보여 준다.
10. 화면 재접속과 heartbeat는 잠수 타이머를 리셋하며, 단순 화면 조회는 리셋하지 않는다.
